# 番茄AI — 产品说明与使用文档

> 将模糊目标，通过AI问答转化为清晰计划，再用番茄钟逐个击破。

---

## 一、产品简介

**番茄AI** 是一款基于番茄工作法 + AI 智能拆解的个人目标管理 Web 应用。

它解决的核心问题是：每个人心中都有渴望达成的目标，但缺乏清晰的执行计划和工作节奏。通过 AI 对话澄清目标、自动拆解为阶段计划、生成每个番茄钟的具体工作事项，帮助你从"想做什么"走向"正在做"。

### 适用人群

- 拖延症、注意力不集中的人群
- 有目标但缺乏执行计划的人
- 希望用结构化方式管理时间的学习者和工作者

### 核心特色

| 特色 | 说明 |
|------|------|
| AI 全流程驱动 | 从目标创建到番茄钟执行，AI 贯穿拆解、规划、事项生成、对话陪伴 |
| 多 AI 提供商 | 支持 DeepSeek、火山引擎、智谱 AI、阿里千问，可随时切换 |
| 优雅降级 | 未配置 API Key 时自动使用本地模拟，功能不中断 |
| 纯前端架构 | 无需后端，数据存储在浏览器本地，隐私安全 |
| 番茄工作法 | 完整的 25 分钟工作 + 休息循环，支持事项完成度追踪 |
| AI 对话操作 | 通过自然语言即可添加、删除、修改番茄钟工作事项 |
| 数据可视化 | 专注趋势图表、目标进度统计 |

## 二、快速开始

### 2.1 启动项目

```bash
# 安装依赖
npm install

# 启动开发服务器（支持局域网访问）
npm run dev

# 构建生产版本
npm run build
```

开发服务器默认运行在 `http://localhost:5173`，同时支持局域网内手机浏览器访问。

### 2.2 配置 AI 模型

首次使用建议配置一个 AI 提供商，以获得完整的 AI 体验：

1. 打开应用，点击右上角「设置」
2. 选择一个 AI 提供商（推荐智谱 AI 或火山引擎，有免费额度）
3. 输入 API Key（仅保存在本地浏览器）
4. 选择模型
5. 点击「测试连接」验证
6. 点击「保存配置」

> 即使不配置 AI，应用也会自动使用本地模拟模式运行，所有功能均可使用。

## 三、功能模块说明

### 3.1 首页仪表盘

**路径**：`/`

首页展示整体概览信息：

- **统计卡片**：今日专注番茄数、累计专注时长、进行中目标数、已完成目标数
- **进行中目标**：卡片展示，含进度条和「开始」入口
- **已完成目标**：低透明度展示历史目标
- **空状态引导**：无目标时引导用户创建

### 3.2 创建目标（AI 对话拆解）

**路径**：`/create`

通过 AI 对话将模糊目标转化为清晰的结果目标：

**操作步骤**：

1. 输入目标名称（必填）和描述（可选）
2. 点击「开始 AI 拆解」
3. AI 逐个提出澄清问题（3-4 轮对话）
4. AI 生成「结果目标 + 成功标准」
5. 确认后自动创建目标，进入计划制定页面

**AI 状态指示**：

- 🟢 绿色 — AI 模型驱动中（显示提供商和模型名）
- 🟡 黄色 — 本地模拟模式（未配置 API Key 或调用失败）

### 3.3 计划制定（AI 阶段规划）

**路径**：`/plan/:goalId`

AI 根据目标自动生成阶段计划：

**操作步骤**：

1. 查看结果目标和成功标准
2. 用滑块设置每日投入时间（1-8 小时）
3. 点击「生成阶段计划」
4. AI 生成 3 个阶段（入门、进阶、精通），每阶段含 3-4 个任务
5. 展开阶段卡片查看任务详情（标题、内容、注意事项、时长）
6. 点击「开始」进入番茄钟执行

### 3.4 番茄钟专注执行

**路径**：`/timer/:goalId`

这是应用的核心页面，包含以下功能：

#### 计时器

- **工作模式**：25 分钟专注
- **短休息**：5 分钟（每完成 1 个番茄后）
- **长休息**：15 分钟（每完成 4 个番茄后）
- **控制按钮**：开始、暂停、跳过、重置

#### 当前番茄钟工作事项

- 进入任务时，AI 自动生成 3 个 25 分钟的工作事项
- 每个事项包含标题和达标标准
- 方形勾选框可标记完成/未完成
- 番茄钟结束时弹出完成弹窗，可标记每个事项的完成状态、完成率和备注

#### AI 对话面板

- 点击右下角浮动按钮展开/收起
- 支持**自然语言指令**操作事项：

| 操作 | 示例指令 | 效果 |
|------|---------|------|
| 添加事项 | `加一个背单词` | AI 识别意图 → 显示确认按钮 → 点击确认后添加 |
| 删除事项 | `删除整理笔记` | AI 查找匹配事项 → 显示确认按钮 → 点击确认后删除 |
| 修改事项 | `修改整理笔记为复习英语` | AI 查找匹配事项 → 显示确认按钮 → 点击确认后修改 |

- AI 会显示初始问候消息
- 所有操作都需要用户确认，防止误操作

#### 整体任务进度

- 展示目标的结果目标和成功标准
- 列出所有任务，包含开始/结束时间和超期标注
- 已超期且未完成的任务显示红色警告
- 当前任务以橙色高亮

#### 4 番茄进度指示器

- 4 个圆点显示当前周期进度
- 每完成 1 个番茄点亮 1 个圆点

### 3.5 统计概览

**路径**：`/stats`

数据可视化页面：

- **统计卡片**：累计番茄、累计专注时长、进行中目标、已完成目标
- **本周专注统计**：最近 7 天每天的专注时长（柱状图）
- **番茄趋势**：最近 7 天每天完成的番茄数（折线图）
- **目标专注统计**：各目标的专注时长、番茄数、进度

### 3.6 设置

**路径**：`/settings`

AI 提供商配置页面：

- **提供商选择**：5 个选项，标注是否有免费额度
- **API Key 输入**：支持显示/隐藏，仅保存在本地
- **模型选择**：根据提供商动态显示可选模型
- **自定义 API 地址**：可选，覆盖默认地址
- **测试连接**：验证 API Key 是否有效
- **配置持久化**：每个提供商的 API Key 和模型分别保存，切换时自动加载

## 四、AI 提供商配置指南

### 4.1 支持的提供商

| 提供商 | 免费额度 | 推荐模型 | 获取 API Key |
|--------|---------|---------|-------------|
| DeepSeek | 无 | deepseek-v4-flash | platform.deepseek.com |
| 火山引擎 | 有 | Doubao-Seed-2.0-Lite | console.volcengine.com/ark |
| 智谱 AI | 有 | GLM-4.7-Flash | open.bigmodel.cn |
| 阿里千问 | 有 | Qwen-Turbo | dashscope.aliyuncs.com |
| 本地模拟 | — | local-mock | 无需配置 |

### 4.2 推荐免费方案

#### 智谱 AI（推荐）

1. 访问 [open.bigmodel.cn](https://open.bigmodel.cn) 注册
2. 在 API Keys 页面创建密钥
3. 复制 API Key 填入设置页面
4. 选择模型 `GLM-4.7-Flash`（免费）

#### 火山引擎

1. 访问 [console.volcengine.com/ark](https://console.volcengine.com/ark) 注册
2. 在「在线推理」页面创建推理接入点，选择免费模型
3. 复制 API Key 填入设置页面
4. 选择模型 `Doubao-Seed-2.0-Lite`

### 4.3 多提供商切换

- 每个提供商的 API Key 和模型配置**分别保存**
- 切换提供商时自动加载该提供商的上次配置
- 当前激活的配置显示在设置页面顶部

## 五、数据持久化

所有数据存储在浏览器 localStorage 中，无需后端服务：

| 数据类型 | 说明 |
|---------|------|
| 目标 | 目标名称、描述、状态、创建时间、截止日期 |
| 阶段 | 阶段规划、进度 |
| 任务 | 任务列表、状态、完成时间 |
| 番茄钟事项 | 当前工作事项、完成状态 |
| 事项完成记录 | 完成率、备注 |
| 专注记录 | 每次专注的时长、日期 |
| 用户偏好 | 每日专注时间、声音/通知设置 |
| AI 配置 | 各提供商的 API Key 和模型 |

**重启浏览器后**：应用会自动恢复所有数据，包括当前进行中的目标、任务和番茄钟工作事项。

## 七、常见问题

### Q: 没有配置 API Key，可以使用吗？

可以。应用默认使用本地模拟模式，所有功能均可使用，只是 AI 生成的结果质量不如真实模型。

### Q: 切换 AI 提供商后，之前的 API Key 会丢失吗？

不会。每个提供商的配置分别保存，切换时自动加载对应配置，切回来时恢复。

### Q: 数据会丢失吗？

不会。所有数据存储在浏览器 localStorage 中，刷新页面或重启浏览器后自动恢复。但清除浏览器数据会导致数据丢失。

### Q: 支持手机使用吗？

支持。应用采用响应式设计，可在手机浏览器中正常使用。开发模式下可通过局域网 IP 访问。

### Q: AI 对话中如何操作事项？

在番茄钟页面，点击右下角浮动按钮打开 AI 面板，输入自然语言指令即可：
- 添加：`加一个背单词`
- 删除：`删除整理笔记`
- 修改：`修改整理笔记为复习英语`

AI 会识别意图并显示确认按钮，点击确认后执行操作。
